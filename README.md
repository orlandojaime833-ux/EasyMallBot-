# 🛍️ EasyMall Bot

Plataforma de vendas no Telegram com IA Gemini e sistema de afiliados.

## 🚀 Funcionalidades

- ✅ Criar loja virtual
- ✅ Adicionar produtos
- ✅ Sistema de vendas
- ✅ IA Gemini para suporte
- ✅ **Sistema de afiliados (10% comissão)**
- ✅ Planos FREE e BASIC
- ✅ Sistema de saques
- ✅ Ranking de afiliados
- ✅ Missões diárias

## 📦 Comandos

| Comando | Descrição |
|---------|-----------|
| `/start` | Menu principal |
| `/criar_loja` | Criar sua loja |
| `/add "Nome" Preço` | Adicionar produto |
| `/produtos` | Listar produtos |
| `/saldo` | Ver saldo |
| `/sacar VALOR` | Sacar saldo |
| `/planos` | Ver planos |
| `/link_afiliado` | Gerar link de indicação |
| `/minhas_comissoes` | Ver comissões |
| `/ranking_afiliados` | Ranking de afiliados |
